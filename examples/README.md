# Physics Examples and Problems

This directory contains worked examples and problems based on academically approved physics.

## Purpose

- Demonstrate practical application of physics principles
- Provide learning exercises
- Show step-by-step problem-solving methods
- Build understanding through calculation

## Guidelines

### For Examples
- Start with clear problem statement
- Show all mathematical steps
- Explain the physical reasoning
- Include units in all calculations
- State assumptions clearly
- Provide final answer with proper significant figures

### For Problems
- Present well-defined scenarios
- Use realistic values and conditions
- Ensure problems are solvable with provided information
- Include solutions (in separate files or sections)

### Academic Standards
- Use standard notation and units (SI preferred)
- Follow established problem-solving methods
- Cite sources for any specialized techniques
- Verify calculations are correct before publishing

## Example Structure

```
examples/
  mechanics/
    free_fall.md
    projectile_motion.md
  thermodynamics/
    heat_transfer.md
```

Each example should be self-contained and educational.
